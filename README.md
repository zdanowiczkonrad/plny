## Options

All formatting options from [`number-currency-format`](https://github.com/zdanowiczkonrad/number-currency-format) are allowed.

Example: 
```
format(29999.99, {
    currency: 'PLN',
    showDecimals: 'NEVER'
}
// '30 000 PLN'
```
